# Church of Sts. Peter & Paul, Banting — Official Web Presence

A modern, responsive multi-page website developed for the Parish of Sts. Peter & Paul in Banting, under the Archdiocese of Kuala Lumpur. This project was designed to provide the local Catholic community with a centralized hub for liturgical schedules, parish history, and ministry outreach.


Technical Stack
Frontend: Semantic HTML5, CSS3 (Custom Variables/Theming)
Icons & Typography: FontAwesome 6.5.0, Google Fonts (Playfair Display & Lato)
Animations: JavaScript Intersection Observer API for scroll-reveal effects
Responsiveness: Mobile-first approach using CSS Flexbox and Grid

Key Features
Dynamic Navigation: A custom-built, glassmorphism-style navbar that transitions on scroll.
Interactive Societies Portal: A filterable-style "Jump Nav" for quickly navigating through 10+ parish ministries.
UX-Focused Contact System: A functional contact form UI with validation logic and a custom "Deo Gratias" success state.
Liturgical Integration: A digital "Weekly Bulletin" preview section designed to bridge the gap between physical and digital parish life.
Thematic Design: A sophisticated "Burgundy & Gold" color palette reflecting the traditional ecclesiastical colors of the patron saints.

Project Structure
index.html` (formerly home.html) — The landing page featuring the Priest's message and the digital bulletin.
about.html` — Detailed parish history timeline and apostolic profiles of St. Peter and St. Paul.
societies.html` — A directory of parish ministries with dedicated call-to-action sections.
contact.html` — Contact portal with integrated Google Maps and location metadata.

Deployment
This site is hosted via [GitHub Pages/Netlify]. The deployment pipeline is connected to this repository for continuous integration; any changes pushed to the `main` branch are automatically updated live.
